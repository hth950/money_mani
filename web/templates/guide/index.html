1. 시스템 개요

Money Mani는 한국(KRX)과 미국(US) 주식을 자동으로 평가해 매매 신호를 Discord로 알려주는 시스템입니다.
단순한 이동평균 하나가 아니라 5가지 관점(기술적 · 펀더멘탈 · 수급 · 인텔리전스 · 매크로)을 종합해 점수를 냅니다.

핵심 흐름
매일 08:00 스캔 → 5축 점수 계산 → 대시보드 표시
장중 08:50~15:35 실시간 모니터 → 60초마다 기술적 신호 감시 → 합의 전환 시 Discord 알림
매시 AI 뉴스 분석 → Intel Score 갱신 → 자동 재스코어링

각 축 상세 설명

📈 Technical Score — 기술적 지표 직접 점수화

계산식: 5개 기술적 지표를 각각 0~1로 점수화한 후 가중 합산

지표가중치기준
RSI 포지션25%RSI 위치에 따라 과매도(높은 점수)~과매수(낮은 점수)
MACD 방향25%MACD > Signal이면 상승, 격차 클수록 높은 점수
이동평균 위치25%현재가가 MA20/60/200 위에 있을수록 높은 점수
볼린저밴드15%하단 근처 = 높은 점수 (반등 기대), 상단 근처 = 낮은 점수
거래량 추세10%최근 거래량이 20일 평균 대비 증가 시 높은 점수

예시: RSI=0.65, MACD=0.70, MA=0.80, BB=0.45, Vol=0.60
→ 0.65×0.25 + 0.70×0.25 + 0.80×0.25 + 0.45×0.15 + 0.60×0.10 = 0.665

상관관계 분석 결과, 5일 단기 수익률과 유일하게 양의 상관(r=+0.033)을 보이는 축입니다.

📊 Fundamental Score — 재무 건전성

데이터 소스

  • 한국: DART 전자공시 API — 연결재무제표(CFS) 우선, 별도재무제표 폴백
  • 미국: yfinance

평가 항목: PER, PBR, ROE, 배당수익률

섹터 상대 평가: 업종 평균 대비 상대적 우열 판단 (Technology PER 기준 35배, 금융업 12배 등)

캐시: 4시간 TTL

💸 Flow Score — 외국인·기관 수급 (KRX 전용)

데이터 소스: 네이버 금융 스크래퍼 (pykrx가 OCI 클라우드에서 차단될 경우 자동 대체)

구성 요소가중치의미
Streak20%연속 순매수 일수
Amount35%순매수 금액 절대값
Ratio25%외국인+기관 합산 비율
Synergy20%외국인·기관 동반 매수 시너지

조회 기간: 14일 / 캐시: 4시간 TTL (장 마감 후 16:10에 강제 초기화)

🧠 Intel Score — AI 인텔리전스 (→ 5장에서 상세 설명)

AI(LLM)가 뉴스·공시를 분석해 종목별 감성 점수를 자동 산출합니다.

캐시: 1시간 TTL

→ 5장 인텔리전스 스코어 상세 설명

🌍 Macro Score — VIX 시장 공포지수

데이터: CBOE VIX 지수 (yfinance, ^VIX)

급격한 점프 없이 연속적으로 변화하도록 구간별 선형 보간(piecewise-linear) 적용

VIX점수상태
≤ 150.80🟢 Calm (안정)
200.70🟡 Caution (주의)
250.50🟠 Elevated (경계)
≥ 350.15🔴 Fear (공포)

전 종목 공통값 / 캐시: 2시간 TTL

복합 점수 계산식 (KRX 예시)

composite = Technical×0.50 + Fundamental×0.10 + Flow×0.20 + Intel×0.10 + Macro×0.10

예시: Tech=0.75, Fund=0.60, Flow=0.70, Intel=0.55, Macro=0.70
→ 0.75×0.50 + 0.60×0.10 + 0.70×0.20 + 0.55×0.10 + 0.70×0.10
→ 0.375 + 0.060 + 0.140 + 0.055 + 0.070 = 0.700 → 매수 추천 ✅

※ 가중치는 8,069건 백필 데이터의 Spearman 상관분석 결과를 기반으로 최적화됨 (2026-03-15)

3. 매수 · 매도 판단 기준

🟢 매수 추천 (BUY)
복합 점수 ≥ 0.65 (65% 이상)
+ 포트폴리오 리스크 한도 내
🟡 관망 (WATCH)
복합 점수 0.40 ~ 0.65
조건을 기다리는 후보군
🔴 매도 추천 (SELL)
복합 점수 ≤ 0.40 (40% 이하)
또는 Exit Score 기준 이하

포트폴리오 리스크 한도 (config/risk.yaml)

최대 보유 종목 수20종목
단일 종목 최대 비중20%
단일 섹터 최대 비중30%
일일 최대 손실 한도-3%

매도 타이밍 스코어 (Exit Scorer)

보유 종목에 대해 별도로 매도 적정성을 평가합니다. 복합 점수와 별개입니다.

구성 요소가중치
추세 (Trend)35%
모멘텀 (Momentum)30%
트레일링 스탑35%
  • Exit Score ≤ 0.25 → 매도 실행 신호
  • Exit Score 0.25~0.40 → 매도 경고
  • 손절 기준: -5%
  • 목표 수익: +15%
  • 최소 보유일: 2일

4. 실시간 모니터 (Realtime Monitor)

일일 스캔이 "아침에 찍어둔 스냅샷"이라면,
실시간 모니터는 장중 연속 감시자입니다.

운영 시간 (자동 시작/종료)

시장시작종료
KRX (한국)평일 08:50 KST평일 15:35 KST
US (미국)평일 22:50 KST익일 06:05 KST

동작 방식

  1. KIS API로 현재가 수집 (실패 시 yfinance 대체)
  2. RollingBuffer에 OHLCV 추가 (최대 200봉 유지)
  3. 검증된 기술적 전략 N개 각각 시그널 계산
  4. 합의 전환(Consensus Flip) 감지: BUY→SELL 또는 SELL→BUY
  5. 합의 전환 시 Discord 알림 전송 + 즉시 재스코어링

합의(Consensus)란?

예: 검증된 전략 8개 중 6개 이상이 BUY 신호 → BUY 합의
→ 이후 5개 이하로 떨어지면 → 합의 전환(Flip) 발생 → Discord 알림

합의 전환 시 Technical Score를 근사값으로 즉시 갱신:
  • BUY 합의 → Technical Score = 0.75
  • SELL 합의 → Technical Score = 0.25

쿨다운: 동일 종목 30분 내 중복 알림 억제

수동 제어: /monitor 페이지에서 ON/OFF 가능

5. 인텔리전스 스코어 (Intel Score)

인텔리전스란?

AI(LLM, Gemini)가 뉴스·공시 텍스트를 읽고 "이 뉴스가 해당 종목에 호재냐 악재냐"를 자동으로 판단해 점수를 내는 기능입니다.
인간이 뉴스를 읽는 과정을 자동화한 것입니다.

인텔 스캔 주기

시장스캔 시간횟수
KRX평일 09:00~15:00 매시 정각하루 7회
US평일 23:00~익일 06:00 매시 정각하루 8회

매 스캔 완료 직후 전 종목 자동 재스코어링이 실행됩니다.

스캔 → 스코어 흐름

  1. 네이버 검색 API로 감시 종목 관련 최신 뉴스 수집
  2. LLM이 각 뉴스 분석:
    • 관련 종목 추출
    • 감성 판단 (POSITIVE / NEGATIVE / NEUTRAL)
    • impact_score (0~1), confidence (0~1)
    • category (earnings / regulation / macro / technical 등)
  3. DB(market_intel_issues) 저장 → 시장 인텔 페이지에서 확인 가능
  4. IntelScorer가 종목별 점수 산출:
    • 최근 7일 이슈 조회
    • 시간 감쇠 적용: 0.85^(경과일수) — 오래된 뉴스는 가중치 자동 감소
    • 카테고리별 과거 정확도 보정 — 틀린 카테고리는 신뢰도 낮춤

정확도 피드백 루프

시스템이 스스로 학습합니다

매일 16:00 → IntelPriceTracker가 시그널 발생 종목의 당일 종가 기록
예측 방향과 실제 가격 변동 비교 → accuracy_score DB 저장
IntelScorer가 카테고리별 과거 정확도를 조회해 신뢰도 낮은 카테고리 가중치 자동 감소

매주 일요일 09:00 → CorrelationReport가 스코어-수익률 Spearman 상관분석 결과를 Discord로 전송

7. 재스코어링 (Rescore) 시스템

왜 필요한가?
일일 스캔은 아침에 한 번만 실행됩니다. 하지만 수급·뉴스·VIX는 장중에도 계속 바뀝니다.
재스코어링은 최신 캐시 데이터를 이용해 Technical을 제외한 4개 축(Fundamental/Flow/Intel/Macro)을 재계산하고 복합 점수를 갱신합니다.

재스코어링이 발생하는 4가지 경우

트리거시점대상
스케줄 재스코어링09:30 / 11:30 / 13:30 / 15:30오늘 스캔된 전 종목
인텔 스캔 완료 후매시 인텔 스캔 직후전 종목
수급 재스코어링16:10 (Flow 캐시 강제 만료 후)전 종목
합의 전환 즉시실시간 모니터에서 Consensus Flip 감지 즉시해당 종목만

합의 전환 즉시 재스코어링은 장중 가장 빠른 업데이트입니다.
실시간 모니터 → 전략 합의 BUY→SELL 전환 감지 → 해당 종목만 즉시 재계산 → 대시보드 반영

8. Paper Trading & 성과 검증

Paper Trading이란?

실제 돈을 쓰지 않고 가상 매매를 시뮬레이션하는 기능입니다.
시스템이 BUY 신호를 내면 가상으로 매수 기록하고, SELL 신호에 가상 매도합니다.

참고: 포트폴리오·신호 대시보드에 "보유중"으로 표시되는 종목은 실제 계좌가 아닌 DB의 가상 포지션입니다. KIS API 실계좌 연동 시에만 실제 매매가 발생합니다.

성과 검증 흐름

  1. BUY 신호 발생 → signal_price 기록 (시그널 당시 가격)
  2. 매일 16:00 IntelPriceTracker → 당일 종가 vs signal_price 비교 → pnl_pct 계산
  3. scoring_results ↔ signal_performance JOIN → 각 축별 Spearman 순위상관계수 계산
  4. 매주 일요일 → 주간 상관분석 리포트 Discord 전송

상관계수 해석

Spearman r의미
0.3 이상✅ 해당 축이 수익률과 양의 상관 → 가중치 유지 또는 확대
0.1 ~ 0.3⚠️ 약한 상관 → 모니터링 필요
0.1 미만❌ 상관 없음 → 가중치 축소 검토
음수🚨 역효과 → 즉시 검토

→ 성과 분석 페이지에서 실시간 확인 가능
