📖 Money Mani 매매 가이드

이 페이지는 Money Mani 시스템을 활용해 매수/매도 판단을 내리는 방법을 설명합니다.

① 시스템 구조 한눈에 보기

매일 장 마감 후 (15:30~) → 일일 스캔 (101개 전략 × 앙상블 필터) → 매매 대시보드 업데이트 + Discord 알림
장중 (09:00~15:30 KST) → 실시간 모니터 → DB 저장 (합의 55% 전환 시만 Discord 1건)
매주 일요일 09:00 → 스코어-수익률 상관분석 리포트 → Discord

② 매매 대시보드 사용법

  1. 매매 대시보드 (/signals)에서 매수 추천 카드 확인
  2. 카드 클릭 → Scoring 탭에서 5축 세부 점수 확인 (기술/펀더멘탈/수급/인텔/매크로)
  3. Market Intel 탭에서 해당 종목 부정 이슈 없는지 확인
  4. 아래 판단 기준표를 참고해 최종 결정 → 수동 주문

③ 판단 기준표

composite_score conviction 권장 행동
≥ 65% HIGH ✅ 매수 적극 검토
50~65% MED ⚠️ 추가 확인 후 소량 진입 검토
< 50% LOW ❌ 관망
SELL + ≥ 65% HIGH 🔴 보유 중이면 매도 검토

④ Discord 알림 해석법

시간대 알림 종류 의미
장 마감 후 일일 요약 앙상블 통과 종목 목록 → 실제 매매 검토 대상
장중 ⬆️⬇️ 합의 전환 55%+ 전략이 방향 전환 → 참고용 (대시보드 재확인 권장)
장중 ⚠️ 긴급 합의 전환 80%+ 압도적 합의 전환 → 즉시 확인 필요
일요일 09:00 📊 주간 리포트 스코어-수익률 상관계수 → 시스템 신뢰도 지표

⑤ 절대 하면 안 되는 것

  • 개별 전략 신호만 보고 즉시 매매 (앙상블 통과 신호만 유효)
  • composite_score 없는 신호로 즉시 진입
  • 같은 종목에 BUY/SELL이 충돌 상태일 때 진입 (관망 필요)
  • Discord 알림 하나만 보고 판단 (대시보드 종합 확인 필수)
💡 요약 루틴
매일 장 마감 후 Discord 확인 → 신호 있으면 매매 대시보드 접속 → HIGH conviction 종목만 → Scoring + Intel 탭 확인 → 수동 주문
